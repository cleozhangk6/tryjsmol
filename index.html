AF-SEGMENT is a web server that automatically segments a given AlphaFold model into confident sub-structures and provides visualisation of the results in 3D.

Start your calculation by either searching a Uniprot ID to retrieve the AlphaFold model or uploading a structural file that is in the PDB or mmCIF format.

e.g. P43238, P29376, P07664, etc.
